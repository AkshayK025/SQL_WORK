1️⃣ ROW_NUMBER()
Assigns a unique row number to each row within a partition (no ties).

2️⃣ RANK()
Assigns a rank based on ordering. Ties get the same rank, but gaps appear after ties.

3️⃣ DENSE_RANK()
Similar to RANK() but no gaps after ties.

4️⃣ NTILE(n)
Divides rows into n buckets as evenly as possible.
